# Подключение возможностей — ключи и память

Базовый агент работает сразу после установки (текст, файлы, код, напоминания).
Ниже — как включить дополнительные способности. Все ключи кладутся на сервер в
`/home/<имя>/.claude-lab/<имя>/secrets/`.

Когда нужен перезапуск: ключ **Groq** читает сам gateway при старте — после него
нужен `sudo systemctl restart <имя>-gateway`. Ключи **навыков** (Perplexity,
YouTube) подхватываются в момент вызова — перезапуск не нужен, достаточно положить
файл.

Команды ниже выполняй под пользователем агента (`sudo -iu <имя>`) — тогда `~`
указывает на его домашнюю папку. `<имя>` — это имя оператора маленькими буквами
(как назвал при установке).

---

## 1. Голосовые сообщения (Groq Whisper) — почти бесплатно (~10 центов/мес)

Чтобы агент понимал голосовые (в т.ч. на русском и арабском).

1. Зайди на https://console.groq.com, войди (удобно через Google).
2. Слева **API Keys** → **Create API Key** → скопируй ключ (`gsk_...`, показывается один раз).
3. На сервере:
   ```
   echo 'gsk_ТВОЙ_КЛЮЧ' > ~/.claude-lab/<имя>/secrets/groq-api-key
   chmod 600 ~/.claude-lab/<имя>/secrets/groq-api-key
   sudo systemctl restart <имя>-gateway
   ```
4. Отправь боту голосовое — он расшифрует и ответит.

У Groq щедрый лимит; на практике личное использование выходит примерно в 10 центов
в месяц — по сути бесплатно.

---

## 2. Веб-поиск и ресёрч (Perplexity) — платно

Чтобы агент искал в интернете и делал ресёрч по свежим источникам.
Навык `perplexity-research` уже в наборе, нужен только ключ.

1. Зайди на https://www.perplexity.ai/settings/api, оформи доступ к API (нужна карта, оплата по расходу).
2. Создай API key (`pplx-...`).
3. На сервере:
   ```
   echo 'pplx-ТВОЙ_КЛЮЧ' > ~/.claude-lab/<имя>/secrets/perplexity-api-key
   chmod 600 ~/.claude-lab/<имя>/secrets/perplexity-api-key
   ```
   (перезапуск gateway не нужен — навык прочитает ключ при первом же запросе)
4. Проси агента: «поищи в интернете...», «сделай ресёрч по...».

Стоимость: платишь по расходу, не подписка. Модель `sonar` — меньше рубля за
обычный запрос. Даже при активном использовании выходит пара долларов в месяц.
Без ключа агент всё равно отвечает из своих знаний — просто без свежего веб-поиска.

---

## 3. Конспекты видео (YouTube) — своя авторизация

Навык `youtube-transcript` вытаскивает текст из видео (лекции, доклады).
Требует бесплатной регистрации в сервисе транскрипций:
```
node ~/.claude-lab/<имя>/.claude/skills/youtube-transcript/scripts/tapi-auth.js
```
и следуй подсказкам (регистрация по email, ключ сохранится сам). После этого
проси: «сделай конспект этого видео <ссылка>».

---

## 4. Долгая (семантическая) память — опционально, продвинутое

По умолчанию агент помнит через файлы `core/MEMORY.md` и `core/LEARNINGS.md` —
этого достаточно для большинства. Если хочешь, чтобы он помнил *много* и искал по
смыслу (а не по ключевым словам) через разговоры и месяцы — можно подключить
внешнюю семантическую память (например [OpenViking](https://github.com/volcengine/OpenViking)).

Это отдельный сервис: его надо поднять и прописать в конфиг шлюза
(`openviking_url`, `openviking_key_file`). Ему нужен свой ключ OpenAI (для «понимания
смысла») — на практике выходит около 1–2 долларов в месяц. Это уже не «для
новичка» — берись, только если реально упёрся в границы файловой памяти. Для старта
не нужно.

---

## Порядок приоритетов

- Сразу: **Groq** (голос) — ~10 центов/мес, по сути бесплатно, полезно всем.
- По надобности: **Perplexity** (веб, меньше рубля за запрос), **YouTube** (видео).
- Позже и по желанию: **семантическая память** (~1–2 доллара/мес).

Всё опционально — базовый агент полностью рабочий и без этих ключей.
